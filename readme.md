### 摸鱼炒股工具，基于 pyside2 透明窗口可拖动，
### 双击透明窗口隐藏，托盘处点击显示，悬停显示详细信息，
### 配置 参考 stock.json , 属性分别为 股票代码 ，买入价，买入股票数量
```[
    {"stock_code": "0.002167", "buy_price": 9.003, "buy_count": 1700},
    {"stock_code": "0.000338", "buy_price": 18.23, "buy_count": 1000}
]```
